# 港股自選清單（Stock Watch）

一個純前端的港股自選清單與指數追蹤工具，提供即時報價、趨勢提示與自動刷新，支援離線快取與 PWA 安裝。

## 功能特色

- **港股自選清單**：預設常用股票代碼，支援新增/刪除與重設清單。
- **即時報價欄位**：價格、升跌、百分比、昨收、開市、最高、最低、成交額。
- **走勢提示**：
  - 價格跳動箭頭（↑/↓）與日內高低提示。
  - 恒指、國指即時變動與趨勢箭頭。
- **智慧刷新**：依交易時段自動刷新，午休/休市/假期顯示狀態並預告下次開市。
- **排序與欄寬記憶**：支援升/降排序，欄寬調整與設定會保存。
- **PWA 離線快取**：可加入主畫面，常用資源離線可用。

## 使用方式

1. **直接開啟**
   - 以瀏覽器開啟 `index.html` 即可使用。
2. **本機啟動簡易伺服器（建議）**
   - 使用任一靜態伺服器（如 VS Code Live Server、`python -m http.server` 等），再開啟 `http://localhost:PORT`。
3. **新增股票**
   - 在「Add Stock」輸入港股代碼（可輸入 700 或 00700），按下 Add。
4. **設定刷新頻率**
   - 在「Refresh (sec)」輸入秒數，最小 3 秒。
5. **排序**
   - 點擊「Sort: Asc/Desc」切換升降排序。
6. **重設清單**
   - 點擊「Reset Default List」回復預設自選清單。

## 資料來源說明

- 股票與指數資料來自 on.cc 相關即時報價 API。
- 香港假期日曆使用 Nager.Date 公共假期 API（可快取；失敗時使用內建備援）。

## 專案結構

- `index.html`：主頁與 UI 結構
- `styles.css`：版面與顏色樣式
- `main.js`：資料抓取、排序、刷新與互動邏輯
- `sw.js`：PWA Service Worker 快取策略

## 注意事項

- 本專案為純前端工具，需瀏覽器可連網取得即時報價。
- 報價僅供參考，請以官方或券商資訊為準。
